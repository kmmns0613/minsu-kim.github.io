OmicsForge

Integrated omics analysis toolkit for intuitive and reproducible data interpretation.

Introduction

OmicsForge is a modular R-based application for transcriptomics analysis and visualization. It was developed to help experimental researchers move more easily from processed omics data to biologically interpretable results.

Many researchers can generate omics datasets, but translating those data into clear and reproducible biological insights often remains difficult. OmicsForge was built to reduce that gap by providing a structured and intuitive workflow that can be used without requiring coding, while still maintaining analytical transparency.

The platform focuses on clarity, reproducibility, and biological relevance rather than unnecessary technical complexity. It incorporates commonly used transcriptomics strategies, including enrichment analysis, pathway-level scoring, and visualization, in a modular framework that can be directly applied to real research scenarios.

Download & Installation

Download OmicsForge (MacOS)

Currently available for macOS only 

Requirements:
  - macOS
  - R (version 4.5 or later)

How to run:
  1. Download OmicsForge.app.zip
  2. Unzip
  3. Double-click OmicsForge.app

  On launch, a web browser will open automatically to run 
  the application.

If macOS blocks the app:
This warning appears because the app is not signed with an Apple developer certificate.

  1. Open System Settings → Privacy & Security
  2. Scroll down to the security section
  3. Click "Open Anyway"

  Alternative method:
  Right-click → Open → Open

If the app still does not launch, run from Terminal:

  ~/Desktop/OmicsForge.app/Contents/MacOS/run

Make sure R is properly installed and accessible from Terminal.

- This version includes all required packages for immediate use.
- No additional installation is required.
- Tested on Apple Silicon M4 macOS environments.
- First launch may take a few seconds 
- while the environment initializes.

• Release Notes (v1.0)

Overview

First public release of OmicsForge, providing a modular and reproducible framework for transcriptomics analysis and visualization.

New Features

Implementation of core transcriptomics modules including preranked GSEA and ssGSEA, along with visualization tools such as ES plots and volcano plots.

System

Distributed as a standalone macOS application (.app) with pre-installed dependencies, enabling immediate use without additional package installation.

Notes

Requires R (version 4.5 or later). On launch, the application runs locally and opens in a web browser.

Additional modules and expanded visualization features are planned for future releases, with long-term extension toward multi-omics integration.
